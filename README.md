## Cómo cambiar el número de WhatsApp

Abre `script.js` y cambia esta línea:

```js
const WHATSAPP_NUMBER = "";
```

Por ejemplo:

```js
const WHATSAPP_NUMBER = "56912345678";
```

Sin `+`, sin espacios y sin guiones.
